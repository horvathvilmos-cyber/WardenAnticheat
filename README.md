This is the future of the Minecraft anticheat's
Check everything like: Timer, Flight, Baritone etc.
Commands:
  /warden - The main command
  /warden alerts - turn off or on the anticheat alerts
  /warden ban - Ban a player with reason
  /warden kick - kick the player with reason
  /warden reload - reload the config and the plugin
  /warden playerinfo - shows every information for the specific player
  /warden help - show the help menü
  /warden violations - show the violations for the specific player
